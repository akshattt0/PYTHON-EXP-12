## Theory

Data wrangling refers to the process of converting raw and unstructured data into a clean and usable format. In practical scenarios, datasets often contain missing values, inconsistencies, or irrelevant information, which must be handled before analysis.

Pandas is a widely used Python library for data analysis and manipulation. It provides powerful data structures such as **Series** and **DataFrame**, which allow efficient handling of tabular data.

Using Pandas, datasets can be imported from files such as CSV or Excel. Once loaded, operations like viewing data, selecting columns, filtering rows, and modifying values can be performed easily.

Handling missing values is a crucial part of data wrangling. Pandas provides functions to detect missing values and either remove or replace them. Data can also be sorted, grouped, and transformed to make it more meaningful.

In this experiment, data cleaning and preprocessing were performed on two datasets:
- **Student Dataset** – used to manage and clean student-related information  
- **Cars93 Dataset** – used to analyze and clean automobile-related data  

These datasets helped in understanding real-world data handling and transformation techniques.
